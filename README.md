Love — GitHub Pages site

This repository contains a simple static site (HTML/CSS/JS) you can host via GitHub Pages.

How to publish

1. Create a GitHub repository at https://github.com/new (public).
2. Add this repo as `origin` and push:

```bash
git remote add origin https://github.com/<your-username>/<repo>.git
git push -u origin main
```

3. In the repository Settings → Pages, set the source to the `main` branch (root) and save.
4. The site will be published at `https://<your-username>.github.io/<repo>/`.
